#### 流程
{% asset_img gitflow.png git flow %}
Workspace：工作区
Index / Stage：暂存区
Repository：仓库区（或本地仓库）
Remote：远程仓库
#### 基本操作
- 放入暂存区
`git add .`
`git add file`
- 提交到仓库
`git commit -m 'log'`
- 撤销工作区修改
`git checkout -- file`
`git checkout .`
- 暂存区回退到工作区
`git reset HEAD file`
